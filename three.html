    Control flow and loops, arrays and objects:

    • Control flow: Control flow is the order your program is functioned. Example as humans we are wired to get up in the morning, brush your teeth and have breakfast. We do certain things for a certain amount of time before heading out the door. Control flow is the way our program would work based on step by step instructions.
    • Loops: Loops is when you want something to keep going until it hits a certain point. Example the vending machine knows to keep releasing chips until there is none left, or with the lotto the system knows to keep going until it draws out a certain number of balls.
    • Arrays: Arrays are multiple actions grouped into one action.
    • Runs actions for Groups of data

    Functions and why they are useful

    Functions are commands that run anything you want, this here names and sets up any action. This here is useful for when you are wanting to set up any type of function. Function is an action to follow. You can add function over function to achieve your overall function. The best way to describe this here it is an recipe for what you are trying to create. Example, I want to create a search display, I would create a recepi to get this done and this is called a function.
