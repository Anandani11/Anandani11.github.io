CSS Float Property

  • Website layout
  • Float to the right
  • Float Left
  • With image

What is this?

Float is a CSS property you use to create colums and placement of a website without having to use tables

How and when do you use this?

  • You can use this to create a website layout exactly like this one
  • Position your content
  • Wrap your content around your images
  • Instead of creating colums and tables you can use the float property to sytle and create your layouts
Float codes: Float Left and Right: #sidebar { float: left; width: 350px; } Float with image: img { float: left; margin: 0 15px 5px 0; border: solid 1px #bbb; }
